# Pengenalan Command Line Interface (CLI)

Modul ini memberikan pemahaman mendasar mengenai penggunaan *Command Line Interface* (CLI) sebagai metode utama dalam mengelola perangkat jaringan. Berbeda dengan antarmuka grafis (GUI), CLI menawarkan kontrol yang lebih cepat, efisien, dan aman untuk administrator jaringan.

**Tujuan Pembelajaran:**
- Memahami perbedaan fundamental antara shell Linux dan RouterOS.
- Melakukan navigasi struktur direktori dan hierarki menu pada CLI.
- Memanfaatkan fitur bantuan (*help system*) dan *autocomplete* untuk mempercepat eksekusi perintah.
- Mengekstrak informasi dasar sistem dan identitas pengguna melalui baris perintah.
